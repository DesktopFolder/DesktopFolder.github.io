How to Play drAAft

drAAft is a competitive All Advancements mod for 1-4 players. In a game of drAAft, each player uses the web drafting tool (that's where you are right now!) to draft bonuses from a set of pooled choices. If gambits are enabled, players can also optionally select from a pool of risky bonuses with downsides. These are not actual draft picks, and can be selected or deselected freely until the draft ends, at which point choices are locked in.

Playing drAAft requires a mod, as well as a datapack (until automated datapack downloading is implemented). For mod setup instructions, please see the basic setup page.

There are 6 pools of picks. Each player can pick infinitely if it is a single-player draft, can take 2 picks per pool if it is a 2-player draft, and can take 1 pick per pool in 3- and 4-player drafts.

Initial pick order is random for the first round, and snake order for following rounds. For example, in a draft with Snakezy, Feinberg, and CroPro, the initial random draft order could be Feinberg > CroPro > Snakezy. In this case, the 4th pick would be Snakezy, then CroPro, and then Feinberg would pick twice, and then CroPro would pick again, and so forth.
